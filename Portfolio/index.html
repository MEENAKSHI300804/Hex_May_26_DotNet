Project

FastXBusBooking

FastXBusBooking is a Full Stack Bus Booking Management System developed using ASP.NET Core Web API, Angular and SQL Server. The application provides secure authentication, seat reservation, payment management and booking history with role-based access.

Key Features

  • User Registration & Login
  • JWT Authentication & Role Based Authorization
  • Search Bus by Source, Destination & Journey Date
  • Real-time Seat Booking
  • Booking History
  • Ticket Cancellation
  • Online Payment Module
  • Admin Management
  • Repository Pattern Architecture
  • Global Exception Middleware
  • xUnit Unit Testing
